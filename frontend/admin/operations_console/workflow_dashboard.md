# Workflow Dashboard

## Features

- Running Workflows
- Failed Workflows
- Retry Queue

## UI Fields

| Workflow ID | Workflow Name | Status | Current Step |
|---|---|---|---|
| WF001 | Leave Approval | Running | Manager Approval |
| WF002 | Payroll Run | Failed | Validation |
| WF003 | Email Notification | Retry | Tool Execution |

## Purpose

This dashboard tracks enterprise workflows and allows admins to identify failures and retries.
