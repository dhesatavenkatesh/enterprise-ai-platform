# Approval Dashboard

## Features

- Pending Approvals
- Completed Approvals
- Escalations

## UI Fields

| Approval ID | Type | Status | Approver |
|---|---|---|---|
| APR001 | Leave Approval | Pending | Manager |
| APR002 | Payroll Approval | Completed | Admin |
| APR003 | Document Approval | Escalated | HR |

## Purpose

This dashboard helps admins and managers review approval queues and escalated requests.
