# Agent Dashboard

## Features

- Running Agents
- Agent Health
- Active Sessions

## UI Fields

| Agent | Status | Health | Active Sessions |
|---|---|---|---|
| HR Agent | Running | Healthy | 5 |
| Payroll Agent | Idle | Healthy | 2 |
| Knowledge Agent | Running | Healthy | 8 |

## Purpose

This dashboard helps admins monitor active AI agents, their health, and current user sessions.
